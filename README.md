# Ovex Express

Aplicación web simple con Express.js que contiene un endpoint "Hello World!".

## Instalación

```bash
npm install
```

## Uso

Para iniciar el servidor:

```bash
npm start
```

El servidor se ejecutará en `http://localhost:3000`

## Endpoints

- **GET** `/helloworld` - Devuelve "Hello World!"

## Ejemplo

```bash
curl http://localhost:3000/helloworld
```

Respuesta:
```
Hello World!
```
